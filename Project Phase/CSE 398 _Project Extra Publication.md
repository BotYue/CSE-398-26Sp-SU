# CSE 398 Project Extra Publication Opportunities

For your completed project, you can have some extra publication opportunities.

### 1. Students Conference

Every year, there will be some student conferences that hosted by professional societies such as IEEE, ASEE.

Some student conferences accept student publications. Example papers in ASEE:

  * [https://peer.asee.org/arduino-esp32-based-smart-irrigation-system ](https://peer.asee.org/arduino-esp32-based-smart-irrigation-system)

  * [https://peer.asee.org/ai-driven-wildfire-detection-with-integrated-air-quality-and-machine-learning-vision-systems-powered-by-an-accelerator-for-early-action](https://peer.asee.org/ai-driven-wildfire-detection-with-integrated-air-quality-and-machine-learning-vision-systems-powered-by-an-accelerator-for-early-action)

  * [https://peer.asee.org/a-yolo-based-semi-automated-labeling-approach-to-improve-fault-detection-efficiency-in-railroad-videos](https://peer.asee.org/a-yolo-based-semi-automated-labeling-approach-to-improve-fault-detection-efficiency-in-railroad-videos)

  * [https://peer.asee.org/ai-driven-wildfire-detection-with-integrated-air-quality-and-machine-learning-vision-systems-powered-by-an-accelerator-for-early-action](https://peer.asee.org/bridging-the-accessibility-gap-designing-a-compact-cost-effective-efficient-braille-printer)

  * [https://peer.asee.org/the-mind-controlled-wheelchair](https://peer.asee.org/the-mind-controlled-wheelchair)

If you want to publish in these conferences, you need to do some extras in Result Analysis of your project, espescially **quantitative results**. You can take a look at the example papers and see how they work.

If you want to get update on which conference accepts submission or need any help, please contact Yue Cao.
